# 🚀 AI PDF Summarizer

An AI-powered full-stack web application that allows users to upload PDF documents and instantly generate intelligent summaries using modern AI models.
The platform provides a secure authentication system, interactive dashboard, and downloadable summaries with a clean user experience.

Built with the MERN stack and designed with scalability and production-ready architecture in mind.

---

## ✨ Features

* 🔐 Secure JWT Authentication (Login / Signup)
* 📄 Upload and process PDF documents
* 🤖 AI-generated smart summaries
* 📊 User dashboard with summary history
* 🎯 Interactive summary slider UI
* ⬇️ Download summaries as PDF
* 🧭 Protected routes with authentication
* ⚡ Modern responsive UI with Tailwind CSS
* 🍪 Cookie-based auth support
* 🧠 RESTful backend architecture

---

## 🛠️ Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Multer (File Upload)

### AI Integration

* AI API for text summarization

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ai-pdf-summarizer.git
```

Install dependencies:

```bash
cd backend
npm install

cd ../frontend
npm install
```

Run project:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_url
ACCESS_TOKEN_SECRET=your_secret
REFRESH_TOKEN_SECRET=your_secret
```

Frontend `.env`:

```
VITE_BACKEND_URL=http://localhost:5000
```
